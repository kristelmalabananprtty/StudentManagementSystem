# 🎓 Student Management System

## 📌 Overview
A Java-based Student Management System that allows users to 
manage student records using a graphical interface.

## Technologies Used
- Java
- JavaFX (GUI)
- JDBC (Database Connectivity)
- PostgreSQL (Database)
- Maven (Build Tool)
- SceneBuilder (UI Design)
